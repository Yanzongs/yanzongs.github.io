Book - 《兄弟》简评

2023-02-04
BOOK
  • 感悟
  • 生活
  • 阅读
N 人看过

余华[著]

瑞士《时报》2000年至2010年来全球最为重要的15本书


前前后后读了其他书友很多的评语，自己却始终不敢落笔。

有好的，夸怪诞的现实极富悲喜张力云云；有坏的，贬无脑的转变与荒唐的逻辑云云；亦有好有坏，称赞每一处妙笔且指出每一处不足。

读完这本书，我的第一想法是：“一个好的故事，一波到底需要几折？”就像是在故事的最后，即使林红已经摇身变成冷漠的老鸨、李光头将带着宋刚的骨灰飞向太空、周游和苏妹过上了美满的生活，可停在这里，我仍然觉得不够，这个故事应该还要再有波折，比如林红对李光头开始的报复、比如李光头几年来污遭勾当的败露、比如周游这个骗子被曾经那些受害者讨伐等等等等，我深信继续写下去，会更吸引我看、会更加令我痛哭，但如果波折太多，故事反而就不是故事了。

停在了这里，细细回想，恰如其分。

第一折，以宋凡平惨死车站，划定了李光头与宋刚旧时代的落幕，迎来的是独立、是成长、是自省，是一切都要靠自己走的路，后面的路怎么走，余华给了我们答案。李光头与宋刚截然不同的个性、选择、态度与结局，看似风光的李光头一路高歌成人，而为了脸面功夫的宋刚却活的近乎不人不类。

可随着宋刚的卧轨自杀，第二个波折到来，惊醒了故事中的每一个人，也惊醒了读者。到头来，李光头是世界的世界英雄，却是自己生活里的生活禽兽；宋刚是世界的世界弃婴，却是自己生活里的生活巨人。你怎么看世界、看自己，你就怎么看李光头与宋刚，你都能从他们身上得到自己满意的答案。

宋刚的死令故事的世界线极速转变，进入了忏悔、罪过、追忆的未来。未来怎么走，余华没有落笔，他留给我们去想象。将死之人如何预测自己的未来？为功、为过、为得、为失，一切从过去中来。

林红在宋刚死后会如何过完剩下的一生，要从她的从前来；李光头在宋刚死后会如何看待争议的一生，要从他的从前来；镇上众生在宋刚死后会如何继续混沌的一生，要从他们的从前来。

我（读者）在宋刚死后会如何评价所有的收束？要从从前来。于是在惊叹、在惋惜、在悲愤、在哀怨、在忏悔、在祷告、在振作、在平复、在恻隐，在一切一切里，又一头扎回到从前去了，这可能也是余华的高明，你要知道未来怎么来，你就要把过去读很多很多遍、读很多很多遍。
